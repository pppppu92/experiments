料金について

基本料金はいくらですか？ +

平日の日中（6:00〜22:00）は2,000円/時間、土日祝は2,500円/時間です。早朝・夜間は割増料金となります。最低3時間からのご利用となります。詳しくはサービス・料金ページをご確認ください。

支払い方法は何がありますか？ +

現金、銀行振込、PayPayでのお支払いが可能です。定期利用の場合は月末締めの翌月払いも承ります。領収書の発行も可能です。

キャンセル料はかかりますか？ +

前日までのキャンセルは無料です。当日キャンセルは基本料金の50%、無断キャンセルは100%となります。お子様の急な体調不良の場合は、状況に応じて配慮いたします。

兄弟・姉妹がいる場合の料金は？ +

2人目のお子様からは基本料金の50%となります。3人目以降も同様です。例：1人目2,000円/時間、2人目1,000円/時間の計3,000円/時間となります。

予約・利用について

どのくらい前から予約が必要ですか？ +

定期利用の場合は1ヶ月前、単発利用は1週間前までのご予約を推奨しています。ただし、スケジュールが空いていれば当日でも可能な限り対応いたします。まずはお電話でご相談ください。

最短・最長でどのくらいの時間利用できますか？ +

最短3時間からご利用いただけます。最長は特に制限はありませんが、宿泊を伴う場合は事前にご相談ください。24時間対応も可能です。

対応エリアはどこまでですか？ +

東京都内全域に対応しています。多摩地区も含みます。交通費は実費でご請求いたします。エリア外の場合もご相談に応じますので、お気軽にお問い合わせください。

定期利用の割引はありますか？ +

週3回以上の定期利用の場合、基本料金から10%割引いたします。月契約の場合はさらにお得な料金設定も可能ですので、詳しくはご相談ください。

安全性について

どのような資格をお持ちですか？ +

保育士資格、幼稚園教諭一種免許、救急法基礎講習修了、チャイルドマインダー、食育インストラクターの資格を保有しています。また、賠償責任保険にも加入しており、万が一の際も安心です。

保険には加入していますか？ +

賠償責任保険（1億円）、傷害保険（500万円）、自動車保険（対人・対物無制限）に加入しています。お預かり中の事故やケガに対して適切な補償をいたします。

緊急時の対応はどうなっていますか？ +

24時間緊急連絡先（090-1234-5678）を設けています。重篤な症状の場合は救急車を呼び、同時に保護者様にご連絡します。軽微な症状でも必ず保護者様に報告し、指示を仰ぎます。

感染症対策はどうしていますか？ +

訪問前の検温・体調確認、マスク着用、手洗い・消毒の徹底、室内換気、おもちゃの消毒などを行っています。シッター自身に症状がある場合はサービスを休止します。

サービス内容について

どのような年齢のお子様まで対応していますか？ +

0歳（生後1ヶ月）から小学6年生まで対応しています。年齢に応じた適切なケアを心がけており、発達段階に合わせた遊びや学習サポートも行います。

病気の子どもでも見てもらえますか？ +

軽度の風邪症状であれば、医師の診断書をご用意いただければ対応可能です。ただし、感染症（インフルエンザ、ノロウイルス等）の場合はお断りさせていただく場合があります。まずはご相談ください。

送迎サービスはありますか？ +

保育園・幼稚園、習い事、病院などへの送迎サービスを行っています（+500円/片道）。自家用車での送迎も可能で、チャイルドシート完備、自動車保険加入済みです。

家事もお願いできますか？ +

お子様に関連する家事（食事準備、食器洗い、おもちゃの片付け、洗濯物取り込み等）は追加料金（+300円/時間）で承ります。お子様の安全が最優先のため、危険を伴う作業はお受けできません。

食事の準備はしてもらえますか？ +

離乳食や幼児食の準備を承ります（+800円/回）。アレルギーにも対応可能です。事前にお子様の好み、アレルギー情報、普段の食事内容をお教えください。食育インストラクターの資格も活かします。
